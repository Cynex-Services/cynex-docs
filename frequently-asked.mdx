## General

### What is Cynex?
Simplify your server with Cynex’s powerful moderation tools and comprehensive guides for seamless management.

### Is Cynex free to use?
Yes! Cynex is free to use with many core features. There are additional premium features available if you choose to upgrade. (Coming Soon)

## Setup and Commands

### How do I set up Cynex in my server?
1. Invite Cynex to your server.
2. Make sure to give Cynex the appropriate permissions.
3. Use `/config` to start the initial configuration.

### Where can I find a list of commands?
You can find a complete list of commands by using the `/help` command in Discord, or by visiting the commands section on our website.

## Troubleshooting

### Cynex isn’t responding to commands. What should I do?
1. Make sure Cynex has the necessary permissions.
2. Check the Status Page to see if there are any outages.
3. Ensure you’re using the correct command prefix if custom prefixes are enabled. (Coming Soon)

### How can I report a bug or issue?
You can report bugs directly through our Support Server for live help.
